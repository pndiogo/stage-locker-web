# Stage Locker Web

## Description

The Stage Locker Web is the browser frontend for the Stage Locker App.

## Installation

To install the dependencies, run:

```bash
npm install
```

## Running the Application

To start the development server, use:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```
